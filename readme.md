# Kerbal.de API v1

Simple API 

## Endpoints

#### /shoutbox (GET and POST)
Use the time_since parameter to retrieve newer posts. Limit is 1000 entries per request.

Filters: 
```
since_time=<timestamp>
```
```
fromApi=<0/1>
``` 
0 = Entry generated by WBB (Shoutbox)
1 = Entry generated by API
